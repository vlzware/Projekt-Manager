# Projekt-Manager

A centralized system for consolidation, control, and viewing of data and processes in small, German-speaking "Handwerker" (tradesman) companies.

## Status

**Iteration 0 — Setup**: Defining the walking skeleton, clarifying details with stakeholders, producing specifications and prototypes. No implementation yet.
